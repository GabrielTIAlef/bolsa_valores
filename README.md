# Bolsa de Valores

Projeto completo de análise de dados financeiros desenvolvido para acompanhamento do desempenho histórico de ações da NASDAQ, utilizando Python, Power BI e análise temporal para apoiar decisões estratégicas de investimento.

A solução foi construída com foco em visualização analítica, storytelling dinâmico e indicadores financeiros automatizados, permitindo transformar dados históricos do mercado em insights claros e acessíveis para investidores e analistas.

---

## Objetivos do Projeto

Este projeto foi desenvolvido com foco em:

- Analisar o desempenho histórico de ações
- Identificar tendências de mercado
- Avaliar comportamento de valorização e desvalorização
- Monitorar volume negociado
- Gerar insights automáticos via narrativa inteligente
- Facilitar análises temporais de investimento
- Simular ambiente analítico para investidores
- Transformar dados financeiros em informações estratégicas
